# DEC-021 双驱动板布局与散热策略决策

- 决策状态：已确认
- 所属阶段：P1
- 日期：2026-04-19

## 1. 背景

P1 需要明确硬件方案与采购，缺失硬件中包含电机驱动板。[7][20]  
当前底盘采用四轮差速结构，需要为四个电机提供底层驱动实现。

## 2. 决策内容

确定采用以下方案：

1. 采用两块双路电机驱动板；
2. 驱动板安装在下层动力区；
3. 驱动板布置应尽量靠近电池与电机线束出口；
4. 当前阶段预留自然散热空间，不额外增加主动散热。

## 3. 决策理由

1. 两块双路驱动板能够直接覆盖四个电机的驱动需求。
2. 下层动力区布局有利于缩短高电流线路。
3. 自然散热方案更简单，符合迭代一快速落地原则。
4. 后续若实测发热明显，再补充散热片或导风。

## 4. 备选方案与放弃理由

### 方案A：单块更大驱动板
当前不采用。  
原因：当前已明确使用两块双路驱动板，更符合现有选型状态。

### 方案B：初期即上主动散热
不采用。  
原因：当前阶段先保留自然散热空间即可，避免过度设计。

## 5. 影响

1. 下层板需为两块驱动板预留安装位与散热间距。
2. 动力支路走线方案需围绕驱动板位置展开。
3. P2 阶段需重点观察驱动板温升表现。

2. 在 `power_architecture.md` 中保持动力支路定义一致。
