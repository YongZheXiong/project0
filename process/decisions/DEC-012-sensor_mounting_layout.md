# DEC-012：传感器安装布局

- 状态：已确认
- 阶段：P1
- 日期：2026/4/19

## 背景
P1 系统架构设计要求明确传感器安装位置、底盘结构布局和接线方案。
项目0在迭代一中需要完成 Mid360 与 D435 的接入，并支撑后续建图、定位、调试和记录。

## 决策
1. Mid360 安装在车体顶部，尽量接近几何中心，并尽量高 于 Orin、电池、壳体等遮挡物；
2. D435 作为前向辅助传感器安装，不承担迭代一主 SLAM 传感器角色。

## 理由
1. Mid360 是迭代一关键感知输入，传感器布局直接影响后续建图与定位效果。
2. D435 在迭代一主要承担辅助感知、记录和调试支撑，深度增强利用属于后续增强方向。
3. 顶部居中的 Mid360 安装有利于减少遮挡、稳定标定关系；
4. D435 前向安装有利于场景可视化、记录与弱光感知辅助。

## 影响
1. 硬件架构中需预留顶部雷达支架与前向相机安装位；
2. 标定文档需覆盖雷达与相机安装外参；
3. 传感器接线与补光位置需围绕该布局展开。
