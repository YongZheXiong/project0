# DEC-013：电源架构主边界

- 状态：确认
- 阶段：P1
- 日期：2026/4/24

## 背景
P1 系统架构设计要求输出电源架构，内容包括电池到各设备供电路径、电压转换方案和电源保护。
电源架构属于系统架构的重要组成部分，直接服务于硬件方案与采购。

## 决策
确定采用：
1. 主电池作为统一能源入口；
2. 动力支路采用 3S 电池直供电机驱动板；
3. 低压侧至少划分为：
   - 计算支路；
   - 传感器/控制支路；
4. Orin NX 使用独立高优先级稳压供电支路，与动力支路电气隔离；
5. 急停只切断动力支路，不切断计算与感知支路。

## 理由
1. P1 要求电源路径与保护边界清晰。
2. Orin NX 是主计算平台，需保证稳定供电.
3. 动力与计算分离有利于降低干扰与故障扩散；
4. 急停只切动力支路，有利于保留运行状态、日志记录与后续处置能力；
5. 符合平台化、单人可维护、可扩展的工程组织方式。

## 影响
1. 电源架构文档将按动力支路、计算支路、传感器/控制支路展开；
2. 采购中需要单独考虑稳压模块与电源保护件；
3. 接线图需体现不同支路分区。
