# DEC-017：传感器安装策略决策

- 状态：确认
- 阶段：P1
- 日期：2026/4/19

## 背景
P1 需要在硬件架构中明确传感器安装位置。 
项目0迭代一中，Mid360 是主空间感知输入，D435 是前向辅助视觉/深度感知输入，补光模块用于弱光视觉支撑。

## 决策
确定采用以下安装策略：
1. Mid360 安装在车体顶部，尽量接近几何中心；
2. Mid360 支架允许明显高于上层板；
3. Mid360 安装优先保证视场，必要时牺牲一定整车紧凑性；
4. D435 安装在车头前向区域，作为前向辅助传感器；
5. 补光模块布置在 D435 附近，但不强求与 D435 做结构一体化。

## 理由
1. Mid360 是迭代一主感知器件，视场与本体遮挡直接影响后续建图与定位效果。
2. D435 在迭代一中主要用于辅助感知、可视化和数据记录，不承担主 SLAM 输入。
3. 补光模块的目标是先支撑弱光视觉能力成立，而不是优先做复杂集成结构。
4. 该方案符合“先平台成立，再逐步增强”的总体推进方式。[7]

## 影响
1. 顶部支架需要优先为 Mid360 预留稳定安装位。
2. 车头前部需预留 D435 与补光模块安装空间。
3. P3 阶段需围绕该布局完成外参标定。
