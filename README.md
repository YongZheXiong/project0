# Project0 — 地面移动具身智能平台

## 项目简介
面向教学楼室内走廊场景的具身智能小车平台，具备感知—定位—导航—语义理解—安全保护闭环能力。

## 项目状态
当前阶段：P1 / 版本：V0.1

## 硬件平台
- 计算：Jetson Orin NX 16GB
- 下位机：STM32F407ZGT6
- 激光雷达：Livox Mid-360
- 深度相机：Intel RealSense D435
- 底盘：四轮差速（GB37-555 ×4）

## 快速开始
1. 环境部署：见 `docs/04_deployment/`
2. 编译：`colcon build`
3. 启动：`bash scripts/start_all.sh`
